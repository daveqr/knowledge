# Intellij Shortcuts

* Run: Shift + F10
* Format: Alt + Cmd + L
* Format Whole File: Shift + Alt + Cmd + L
* Code Completion: Ctrl + Space
* Comment: Cmd + /
* Go to Definition: Cmd + B
* Recent Files: Cmd + E
* Recent Locations: Shift + Cmd + E
* Previous: Alt + Command + Arrow
* Delete Line to Clipboard: Alt + X
* Delete Line: Cmd + Y
* Refactor/Rename: Shift + F6
